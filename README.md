# Premium GTA RP Website

Features:
- Rockstar Games style
- Neon / Phonk design
- Animated loading screen
- Background video
- Discord webhook form
- Mobile adaptation
- GTA Online UI style

Deploy:
1. Upload all files to GitHub repository
2. Open Settings -> Pages
3. Enable GitHub Pages

Discord webhook:
Open index.html
Find:
PASTE_DISCORD_WEBHOOK_HERE

Insert your Discord webhook URL.
